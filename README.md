# phishing-email-detection
NLP binary classification of scam phishing emails using BERT.
An individual project done with the University of Southern California's Center for Artificial intellegence in Society student branch.

Please read **writeup.mb** for the project write-up.

Final_training_DisilBert.ipynb includes the code and final results from model training and testing done with the final hyperparameters. 

unifinished_test_training_Distilbert.ipynb includes an earlier, less polished, training run with a maximum token length value of 512 and without data leakage checks.
